ISSUE 2: Chapters Section Overwhelming
Problem: Showing all 27 chapters creates scroll fatigue and visual overload.
Fix: Implement expand/collapse with smart defaults:
Collapsed state (default): Show only:

Current chapter (highest unlocked with < 95% progress)
Next locked chapter (to show what's coming)

Expanded state: Show all chapters in current grid layout
jsx// ChapterCarousel.jsx
const [isExpanded, setIsExpanded] = useState(false);

// Find current chapter (first unlocked with progress < 95%)
const currentChapter = chapters.find(ch => ch.isUnlocked && ch.progress < 0.95);
const currentIndex = chapters.indexOf(currentChapter);

// Collapsed: show current + next
const visibleChapters = isExpanded 
  ? chapters 
  : chapters.slice(currentIndex, currentIndex + 2);

// Add expand button
<button 
  onClick={() => setIsExpanded(!isExpanded)}
  className="text-primary-500 font-medium text-sm"
>
  {isExpanded ? 'Show less' : `View all ${chapters.length} chapters`}
</button>

ISSUE 3: Activity Heatmap Low Contrast
Problem: Light grays on light background are invisible.
Fix: Use stronger color progression:
jsx// ActivityHeatmap.jsx color scale
const getColorClass = (count) => {
  if (count === 0) return 'bg-neutral-200'; // Visible empty state
  if (count <= 5) return 'bg-primary-200';
  if (count <= 15) return 'bg-primary-300';
  if (count <= 30) return 'bg-primary-400';
  if (count <= 50) return 'bg-primary-500';
  return 'bg-primary-600';
};
Also add a subtle border to each cell: border border-neutral-100
Consider adding the legend (Less/More gradient) at the bottom like GitHub does.
